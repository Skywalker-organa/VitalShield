VitalShield :  Secure Healthcare Consent Management System

VitalShield is a secure healthcare consent management platform that enables patients to control access to their sensitive medical data. The system integrates encryption, audit logging, and AI-based risk analysis to ensure privacy, transparency, and secure data sharing between patients, healthcare organizations, and doctors.

Features

• End-to-end encryption using AES-based symmetric encryption (Fernet) to protect medical data
• SHA-256–based audit logging to ensure tamper-proof consent tracking
• AI-based consent risk analysis using NLP techniques such as TF-IDF and cosine similarity
• Role-based dashboards for patients, healthcare organizations, and doctors
• Secure consent lifecycle management including request, approval, rejection, and revocation
• Interactive web interface built using Streamlit

Technologies Used

Languages
Python, SQL

Libraries and Frameworks
Streamlit, Cryptography, Scikit-learn, Pandas, NumPy

Security
AES Encryption, SHA-256 Hashing, Secure Key Management

Database
SQLite

Machine Learning / NLP
TF-IDF Vectorization, Cosine Similarity, Risk Classification
